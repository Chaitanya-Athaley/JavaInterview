Q: what challanges faced when migrate java 1.8 to java 17
Answer:
Common challenges:
Removed or deprecated features
Some old APIs and internal JDK classes are removed or strongly restricted in Java 17.
Third-party library compatibility
We often need to upgrade Spring, Hibernate, Maven plugins, logging libraries, etc.
Build tool upgrade
Maven or Gradle version may need upgrade.
CI/CD pipeline and Docker base image may also need update.
Security and TLS changes , SSL certificate
